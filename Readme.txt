CAROM GAME
~~~~~~~~~~~~~~~~~~~~~~~~~

About:
This is a simple singlePlayer Carom Game.

How to run:
-> The makefile compiles the code. (Type 'make' on the terminal)
-> Now type './carom_game' (on terminal) to run the executable. {linux}

Controls:
-> Use the right arrow and left arrow key to move the striker to the right and the left respectively.
-> Use the Up arrow key and the Down key to increase and decrease the speed respectively.
-> Use the a and c keys to rotate the striker anti-clockwise and clockwise for hitting.
-> Initiate the strike by pressing the Spacebar.
-> Use the tab key to bring the striker back to initial position.
-> Release the right mouse to initiate the strike.
